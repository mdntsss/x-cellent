---

## 🧩 Что такое X-CELLENT?

**X-CELLENT** — это клиентский мод для **Minecraft Java 1.20.1 (Forge)**, который расширяет стандартный интерфейс и добавляет дополнительные инструменты для более комфортной и информативной игры.  
Мод работает **на стороне клиента**, поэтому его можно использовать на большинстве серверов, где разрешены стандартные клиентские модификации.

---

## ✨ Основные возможности

- **Пользовательский клиент** с упором на удобство и читаемость интерфейса.
- **Клиентские функции** без обязательной установки мода на сервер.
- **Визуальные улучшения** и подсказки, помогающие лучше контролировать ситуацию в игре.
- **Гибкая настройка модулей** под свой стиль игры и разные режимы.

---

## 🚀 Установка

1. Установить **Minecraft Java 1.20.1**.  
2. Установить **Forge 1.20.1** (классический клиент Forge-лоадера).  
3. Скачать файл `X-CELLENT.jar`.  
4. Положить `X-CELLENT.jar` в папку `mods` вашего клиента.  
5. Запустить игру через профиль **Forge 1.20.1**.
